Possibilita obtenção de dados sobre este portal de dados abertos no formato de Json com três chaves.

  • sucess: Retorna um boolean true ou false
  • result: Os dados resultantes da requisição
  • help: Uma string de documentação para a função que você chamou

/group_list

Retorna uma lista dos nomes dos grupos do site.
Parametros:

  • order_by(string)o campo para ordenar a lista deve ser 'name' ou 'packages' (opcional)
  • limit(int)se fornecido, a lista de grupos será dividida em páginas de no máximo um limite de grupos por página e somente uma página será retornada por vez (opcional)
  • groups (list of strings)uma lista de nomes dos grupos a serem retornados, se forem fornecidos apenas grupos cujos nomes estejam nesta lista, serão retornados (opcional)

Listar todos os grupos

get http://demo.ckan.org/api/3/action/group_list

result{
0: "acessibilidade"
1: "administração"
2: "educacao"
3: "meio-ambiente"
4: "mobilidade"
5: "saude"
...}

return type: list of strings

/tag_list

Retorna uma lista das tags do portal
Parametros:

  • query(string)Uma consulta de nome de marca para pesquisar, se forem fornecidas apenas as tags cujos nomes contenham essa string serão retornadas (opcional)
  • vocabulary_id (string)O id ou o nome de um vocabulário, se somente as tags que pertencem a este vocabulário forem retornadas (opcional)
  • all_fields (bool)Retornar dicionários de tags completos em vez de apenas nomes (opcional, padrão: False)

Listar todas as tags

get http://demo.ckan.org/api/3/action/tag_list

result{
0: "projetos-de-pesquisa"
1: "administração"
2: "educacao"
3: "linguagens-de-programação"
4: "banco-de-dados"
5: "horários"
...}

return type: list of dictionaries

/package_search

Retorna os metadados dados e metadados de um conjunto de dados (pacote) e seus recursos.
Parametros:

  • id(string)O id ou nome do conjunto de dados

  • include_tracking (bool)adicionar informações de rastreamento ao conjunto de dados e recursos (padrão: False)

/group_show

Retorna os detalhes de um grupo
Parametros:

  • id(string)O id ou nome do grupo
  • include_datasets (bool)incluir uma lista truncada dos conjuntos de dados do grupo (opcional, padrão: False)
  • include_dataset_count (bool)incluir a contagem completa do pacote (opcional, padrão: True)
  • include_extras (bool)incluir os campos extras do grupo (opcional, padrão: True)
  • include_users (bool)incluir os usuários do grupo (opcional, padrão: false)
  • include_groups (bool) incluir os subgrupos do grupo (opcional, padrão: True)
  • include_tags (bool)incluir as tags do grupo (opcional, padrão: True)
  • include_followers (bool)incluir o número de seguidores do grupo (opcional, padrão: True)
