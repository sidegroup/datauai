Possibilita obtenção de dados sobre o dataset de turmas

  • sucess: Retorna um boolean true ou false
  • result: Os dados resultantes da requisição
  • help: Uma string de documentação para a função que você chamou

/Rotas

API URL: http://127.0.0.1:5000/api/3/action/datastore_search?resource_id=f0768119-dcd2-4867-a814-0d54b5c31485

  • resource_id (string)id ou nome do recurso a ser pesquisado (obrigatorio)
  • filtros (dicionário) - condições de correspondência para selecionar, por exemplo, "key1": "a", "key2": "b"} (opcional)
  • q (string ou dicionário) - consulta de texto completo. Se for uma string, ela pesquisará em todos os campos em cada linha. Se for um dicionário como {"key1": "a", "key2": "b"}, ele pesquisará em cada campo específico (opcional)
  • distinct (bool) - retorna apenas linhas distintas (opcional, padrão: false)
  • limit (int)número máximo de linhas a retornar (opcional, padrão: 100)
  • campos (lista ou seqüência separada por vírgulas) - campos a serem retornados (opcional, padrão: todos os campos na ordem original)
  • sort (string) - nomes de campos separados por vírgula com ordenação, por exemplo: “fieldname1, fieldname2 desc”
